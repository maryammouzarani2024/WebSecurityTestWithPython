Simple REST API Documentation

This is a simple REST API example built with Python Flask. All routes are prefixed with /api/users.

Base URL

http://127.0.0.1:5000/api/users

Endpoints

Method Endpoint Description Request Body Response
GET /api/users Retrieve a list of all users None 
[
  { "id": 1, "name": "Alice" , "age": 30 },
  { "id": 2, "name": "Bob" , "age": 25 }
]
GET /api/users/<id> Retrieve a single user by ID None 
{ "id": 1, "name": "Alice" , "age": 30 }
POST /api/users Create a new user

DELETE /api/users/<id> Delete a user None

Usage Example (Python)

import requests

BASE = "http://127.0.0.1:5000/api/users"

# GET all users
users = requests.get(BASE).json()
print(users)

# POST create user
new_user = requests.post(BASE, json={"name": "Charlie", "age": 25}).json()
print(new_user)

# PUT update user
updated = requests.put(f"{BASE}/1", json={"name": "Alice Updated", "age": 30}).json()
print(updated)

# PATCH update user
patched = requests.patch(f"{BASE}/1", json={"name": "Alice Patched"}).json()
print(patched)

# DELETE user
deleted = requests.delete(f"{BASE}/2").json()
print(deleted)

Notes

  • All responses are JSON.
  • Status codes follow REST conventions (200 OK, 201 Created, 404 Not Found, 400 Bad Request).
  • This API uses an in-memory data store; data resets when the server restarts.
